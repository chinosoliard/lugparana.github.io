    Estamos en github

    LUG Paraná 2016-04-14 21:12

    Buscamos una forma de mantenernos al día con las cosas del LUG.

    A raíz de los problemas para compartir los materiales relacionados al LUG, se decidió crear un espacio en github, para así disponer de las cosas cuando sea necesario.
    Tambien, se considera migrar el sitio web a Pelican y hostearlo en github pages.
